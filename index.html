Guide to Updating Menu Code Blocks

How to use this guide: This document shows you how to make common changes to the menu code blocks. Each section includes examples, visual comparisons, and step-by-step instructions.

1. Changing Menu Links

To update where a link goes, find the href attribute and change the URL inside the quotes.

<a href="/change-this-part" style="color: white; text-decoration: none;">Link Text</a>
Before:
<a href="/election-security" style="color: white;">Election Security</a>

Links to: /election-security

Text shows: Election Security

After:
<a href="/new-program-page" style="color: white;">Election Security</a>

Links to: /new-program-page

Text shows: Election Security

Important: Make sure to keep the quotes around the URL and don't add any spaces inside the quotes.

2. Changing Link and Button Text

To change what text appears on the link or button, edit the text between the opening <a> and closing </a> tags.

<a href="/link-url" style="color: white; text-decoration: none;">Change This Text</a>
Before:
<a href="/link" class="learn-more-button">LEARN MORE</a>
LEARN MORE
After:
<a href="/link" class="learn-more-button">ENROLL NOW</a>
ENROLL NOW

3. Changing Colors

Colors are defined using hex codes (like #173852). You can update these to change the background color, text color, or button color.

Changing Background Color:

<div class="menu-name-megamenu" style="background-color: #173852; padding: 20px;">
Before:
<div style="background-color: #173852; padding: 10px; color: white;">Dark Blue Background</div>
Dark Blue Background
After:
<div style="background-color: #0a2240; padding: 10px; color: white;">Darker Blue Background</div>
Darker Blue Background

4. Changing Menu Width

To make the menu wider or narrower, change the width value.

<div class="megamenu-content" style="width: 300px; max-width: 100%;">
Before:
<div style="width: 300px; background-color: #f0f0f0; padding: 10px;">This menu is 300px wide</div>
This menu is 300px wide
After:
<div style="width: 400px; background-color: #f0f0f0; padding: 10px;">This menu is 400px wide</div>
This menu is 400px wide
Note: When changing width, keep the max-width: 100%; part to ensure the menu still works on mobile devices.

5. Adding a New Link

To add a new link to the menu, copy and paste an existing list item (<li>) and change the link and text.

<li style="margin-bottom: 10px;"> <a href="/your-new-page" style="color: white; text-decoration: none; font-size: 14px; display: block; padding: 5px 0; transition: all 0.2s ease;"> Your New Link Text </a> </li>
Before:
<ul style="list-style: none; padding: 0; margin: 0;"> <li><a href="/link1">First Link</a></li> <li><a href="/link2">Second Link</a></li> </ul>
  • • First Link
  • • Second Link
After:
<ul style="list-style: none; padding: 0; margin: 0;"> <li><a href="/link1">First Link</a></li> <li><a href="/link2">Second Link</a></li> <li><a href="/new-link">New Added Link</a></li> </ul>
  • • First Link
  • • Second Link
  • • New Added Link

8. Troubleshooting

If your menu doesn't look right after making changes, check these common issues:

Problem Solution
Menu disappeared completely Check for missing opening or closing tags, especially <div> tags. Make sure all tags have matching open and close pairs.
Text color is wrong Check the color value in the style="color: value" attribute. Make sure it's a valid color (like white or #FFFFFF).
Button looks broken Check that all the button style properties are still there. Don't remove any of the style properties like display: inline-block; or border-radius: 4px;.
Links don't work Make sure the href attribute has the correct URL format. URLs should start with a forward slash (/) or http://.
Menu width is off Check that you didn't remove the max-width: 100%; part, which keeps the menu responsive on mobile.
Final tips:
  • Always make a backup copy of your code before making changes
  • Test your changes on a development site before putting them live
  • When copying and pasting code, make sure to include all opening and closing tags
  • If something breaks, compare your code to the original to find the issue
  • For colors, you can use color picker tools online to find the exact hex code you want
